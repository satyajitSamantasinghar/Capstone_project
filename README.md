# Bash Scripting Suite for System Maintenance

# Objective
Automate common system maintenance tasks such as backup, updates, and log monitoring.

# Tools Used
- Bash scripting
- Git Bash on Windows 11
- Visual Studio Code

### project Files
| Script | Description |
|--------|--------------|
| `backup.sh` | Creates timestamped backups of user data |
| `update_cleanup.sh` | Performs simulated system updates and cleans temp files |
| `log_monitor.sh` | Scans log files for error keywords |
| `maintenance_menu.sh` | Combines all scripts into a single menu-driven tool |

 How to Run
1. Open **Git Bash** in the project folder.
2. Make all scripts executable:
   ```bash
   chmod +x *.sh
